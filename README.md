# ZonkeLabs AI

A simple research landing page for ZonkeLabs and CCOS: Cognitive Context Operating System.

## Stack

- Static HTML
- CSS
- JetBrains Mono / Iosevka-style font stack matching the Annotated project

## Local development

```bash
npm run dev
```

You can also open `index.html` directly in a browser.

No build step is required.

## Assets

- Official logo: `public/logo4.png`
